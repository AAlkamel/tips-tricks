CSS Regular Expressions

Regular expressions (regex) are a powerful tool for matching patterns in strings. In CSS, regex can be used in various contexts, such as attribute selectors and content generation.

Attribute Selectors with Regex

CSS allows you to use regex-like syntax in attribute selectors to match elements based on their attributes. For example:

/* Select elements with a class that starts with "btn-" , e.g. btn-primary or btn-secondary btn-primary-outline etc. */
[class^="btn"] { /* styles */ } [class*="-primary"] { /* styles */ } [class$="-outline"] { /* styles */ } /* etc */

In this example, the regex [^a-zA-Z0-9] matches any character that is not a letter, number, or underscore, the regex [*a-zA-Z0-9] matches any character that is a letter, number, or underscore, and the regex [$a-zA-Z0-9] matches any character that ends with a letter, number, or underscore. These selectors allow you to target elements based on specific patterns in their class names or other attributes.
